# 🎮 恶魔轮盘 - Python命令行版

## ✨ 项目特色

### 🚀 **极致轻量**
- 📦 **单文件实现** - 一个.py文件包含完整游戏
- ⚡ **秒级启动** - 无需加载，直接开玩
- 🐍 **零依赖** - 仅需Python标准库
- 💾 **内存友好** - 运行时<10MB内存占用

### 📚 **代码质量**
- 🏗️ **面向对象设计** - 清晰的类结构和职责分离
- 🎯 **设计模式应用** - 策略模式、状态模式等实际应用
- 🔄 **完整重构案例** - 展示从过程式到面向对象的进化
- 📝 **类型提示完整** - 完整的Python类型注解

### 🎮 **核心玩法**
- ⚖️ **策略深度** - 8种不同功能的道具系统
- 🤖 **智能AI** - 根据局势动态调整策略的对手
- 🎲 **随机与策略结合** - 子弹随机，决策靠智慧
- 🔄 **完整游戏循环** - 装弹、射击、道具、胜负判定

## 🕹️ 快速开始

### 安装与运行
1. 前往 [Release](https://github.com/qiufengcute/eml/releases) 下载最新版本程序
2. 运行下载的程序

### 游戏控制
```
[系统]请选择行动：
1: 射击自己
2: 射击敌人  
3: 使用道具
4: 查看状态
5: 退出游戏
```

## 🛠️ 技术架构

### 核心类设计
```python
Game (游戏主控制器)
├── Player (玩家状态)
├── BulletChamber (子弹系统)
├── PropSystem (道具系统)
└── AIController (AI逻辑)
```
